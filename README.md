# Homebrew Tap

Personal Homebrew tap for rockorager's projects.

## Installation

```sh
brew tap rockorager/tap
```

## Formulas

### prise

Terminal multiplexer targeted at modern terminals.

```sh
brew install rockorager/tap/prise
```

### zigdoc

Command-line tool to view documentation for Zig standard library symbols.

```sh
brew install rockorager/tap/zigdoc
```
